This plugin allows you to add social 'like' and 'share' buttons to your site and individual pages.

== Description ==

Like & Share My Site (LSMS) is a plugin that will allow you to have social 'like' and 'share' buttons added to the end of your posts. Currently it can display buttons for Facebook, Twitter, Google+, Tumblr & StumbleUpon.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `lsms` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. under the plugin section of the admin menu, selecte the Like & Share My Site Options to configure the plugins settings
